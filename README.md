# exporter_card_import_calculator

داشبورد Streamlit فارسی و راست‌چین برای ارزیابی واردات انجام‌شده با کارت بازرگانی صادرکننده، بازسازی‌شده بر اساس مدل نهایی Google Sheet / Excel.

این ابزار هزینه واردات، ارزش افزوده، سه سناریوی مالیات عملکرد، کارمزد مالک کارت، ذخایر ریسک، وثیقه پیشنهادی و تصمیم نهایی را به‌صورت مدیریتی برآورد می‌کند. محاسبات داخل Python انجام می‌شوند و برنامه برای محاسبه زنده به Google Sheets وابسته نیست. نتایج جایگزین نظر مشاور مالیاتی، گمرکی، حقوقی، حسابداری یا بانکی نیستند.

## اجرا

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## ساختار پروژه

```text
.

```

## فرض‌های محاسباتی مهم

- واحد پیش‌فرض مبالغ داخلی تومان است.
- بهای خارجی کالا، حمل و بیمه با نرخ ارز واردشده به تومان تبدیل می‌شوند.
- پایه ارزش افزوده واردات برابر پایه مشمول گمرکی به‌علاوه حقوق گمرکی، سود بازرگانی و سایر عوارض واردات است.
- پایه مشمول گمرکی می‌تواند به‌صورت محاسباتی یا با مبلغ قطعی دستی تعیین شود.
- ارزش افزوده فروش داخلی برای کالای مشمول باید در صورتحساب درج شود؛ تهاتر اعتبار واردات یا صادرات این الزام را حذف نمی‌کند.
- سه سناریوی مالیات شامل مالیات صرفاً بر کارمزد، مالیات سود برآوردی اینتاکد و مالیات سود سفارشی است.
- بهای تمام‌شده کل شامل ارزش افزوده واردات به‌عنوان خروج وجه است؛ مبلغ بدون ارزش افزوده قابل بازیافت نیز جداگانه محاسبه می‌شود.
- کل هزینه برآوردی مجری شامل بهای تمام‌شده، کارمزد مالک کارت، خالص ارزش افزوده پرداختنی و هزینه اداری است.
- تصمیم نهایی یک شاخص مدیریتی است: ریسک بیش از ۷۰ یا منفعت خالص منفی باعث رد معامله می‌شود؛ وجود ریسک بیش از ۲۰ یا وثیقه پیشنهادی مثبت، اخذ تضمین و کنترل تکمیلی را پیشنهاد می‌کند.

## خروجی Excel

فایل نمونه `sample_output.xlsx` با دستور زیر بازتولید می‌شود و یک Workbook فارسی با فرمول‌های بین‌شیتی، هشدار VAT، امتیاز ریسک و تصمیم نهایی است:

```bash
python3 -c "from utils.exporter import generate_sample_output; generate_sample_output('sample_output.xlsx')"
```

شیت‌های فایل نمونه:

- `ورودی‌های معامله`
- `هزینه‌های گمرکی و واردات`
- `محاسبات ارزش افزوده`
- `سناریوهای مالیات عملکرد`
- `چک‌لیست ریسک`
- `محاسبه وثیقه پیشنهادی`
- `خلاصه تصمیم مدیریتی`

شیت نمونه شامل ردیف‌های واقعی برای ارزش واردات، نرخ ارز، HS Code، فاکتور داخلی، هزینه‌های گمرکی، محاسبه VAT واردات و فروش داخلی، سه سناریوی مالیات عملکرد، چک‌لیست ۲۰ موردی ریسک، محاسبه وثیقه و خلاصه تصمیم مدیریتی است. بسیاری از خروجی‌ها با فرمول‌های Excel به شیت‌های دیگر وصل شده‌اند تا مسیر محاسبه قابل پیگیری باشد.

جزئیات تکمیلی فایل نمونه:

- نرخ‌های کارمزد حواله، اسپرد ارز، تعدیل ارزش گمرکی، حقوق ورودی، سود بازرگانی و VAT از سلول‌های نرخ همان شیت خوانده می‌شوند و در فرمول‌ها hardcoded نیستند.
- هزینه اداری/حسابداری و ذخیره مستقیم ریسک صاحب کارت در شیت سناریوهای مالیات عملکرد به‌صورت ورودی آشکار آمده‌اند.
- شیت خلاصه تصمیم مدیریتی سناریوی مالیات منتخب، درصد کارمزد از ارزش واردات، منفعت خالص به ازای هر دلار، تعداد کنترل‌های انجام‌شده/انجام‌نشده، وضعیت پرداخت جداگانه VAT توسط خریدار و توصیه عملیاتی نهایی را نشان می‌دهد.
- سلول‌های بله/خیر در ورودی‌های معامله، اینتاکد و چک‌لیست ریسک دارای dropdown هستند.
- هشدارهای کلیدی درباره درج VAT فروش داخلی، کنترل دسترسی سامانه جامع تجارت و ضرورت وثیقه کافی در Workbook برجسته شده‌اند.
- بخش «ریسک‌های باز مهم» در خلاصه مدیریتی، سه کنترل انجام‌نشده با بیشترین وزن ریسک را به‌صورت خودکار از چک‌لیست استخراج می‌کند.
- اگر VAT فروش داخلی جداگانه از خریدار دریافت نشود، خلاصه مدیریتی هشدار نقدینگی و اثر احتمالی بر سود صاحب کارت یا مجری را نمایش می‌دهد.
- ذخیره مالیات عملکرد در شیت وثیقه بر اساس بدترین سناریوی مالیات عملکرد محاسبه می‌شود، نه فقط سناریوی منتخب.
- بخش «تحلیل حداقل کارمزد قابل قبول» کف پیشنهادی کارمزد هر دلار را برای پوشش مالیات منتخب، هزینه اداری/حسابداری و ذخیره مستقیم ریسک صاحب کارت محاسبه می‌کند.
- توصیه عملیاتی نهایی بر اساس امتیاز ریسک و منفعت خالص صاحب کارت تولید می‌شود.

در تب «خروجی Excel» علاوه بر گزارش کامل، چک‌لیست ریسک و خلاصه تصمیم نیز جداگانه قابل دریافت‌اند.
